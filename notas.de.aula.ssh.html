### Usando Chaves no ssh

# criar a chave do usuário no (cliente)

ssh-keygen

obs.: fornecendo uma passphrase a chave privada será criptografada.

# copiar a chave do usuário para o Servidor

ssh-copy-id usuario@ip_do_servidor

# para acessar o servidor

ssh usuario@ip_do_servidor

# Para trocar a chave de criptografia (passphrase) da chave-privada
 
ssh-keygen -f id_rsa -p
